# Galaxy Dragon 3D Title — fixed build

Upload **all five files** to the root of the GitHub repository:

- `index.html`
- `main.js`
- `glyphs.js`
- `Over There.ttf`
- `README.md`

Then enable GitHub Pages from the `main` branch and `/ (root)` folder.

This version avoids the Troika package that caused the earlier blank page. It builds extruded letter geometry directly from the supplied font outlines and keeps the Hide Menu button independent from the 3D module, so the button still works even when a loading error occurs.
