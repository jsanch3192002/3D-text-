# J.S.R / Speed Dial — Clean 3D

A clean browser-rendered 3D title with individual letter animation.

## Included
- Black extruded 3D text
- `J.S.R` and `Speed Dial` defaults
- Green-screen background by default
- Individual letter animation and simple camera movement
- No glow, bloom, chrome, HDR, particles, or post-processing

## GitHub Pages
Upload these files directly to the repository root:

- `index.html`
- `main.js`
- `glyphs.js`
- `Over There.ttf`
- `README.md`

Do not upload the containing folder itself. Commit the files, wait for Pages to deploy, then refresh the site in a private tab if Safari shows an older cached version.

The page loads Three.js and SVGLoader from jsDelivr, so an internet connection is required when opening it.
