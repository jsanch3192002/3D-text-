# Galaxy Dragon Chrome 3D Title

Upload these files to the root of your GitHub repository:

- `index.html`
- `main.js`
- `Over There.ttf`
- `README.md`

Then enable GitHub Pages from the `main` branch and `/ (root)` folder.

The project uses Three.js and opentype.js from jsDelivr, so the device needs internet access when opening the page.
